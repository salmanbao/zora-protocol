# Introduction

#### Imagination Onchain

The Zora Network is a fast, cost-efficient, and scalable Layer 2 built to help bring media onchain.
Many L2s are currently DeFi centric whereas the Zora Network is an ecosystem that puts NFTs first. 

- Transactions confirm in seconds
- Minting an NFT costs less than $0.50
- Opens up new possibilities onchain

Secured by Ethereum, Powered by the [OP Stack](https://stack.optimism.io/).
Although Zora deployed the Zora Network software, Zora does not operate the Zora Network itself.
